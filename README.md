# JMSort
瞎搞弄的排序算法。


###大概思路就是两两比较之后每四个进行一次比较，最后将得到的最大的数放置在数组末尾，剩下的继续两两比较。因为上次比较的数据是可以复用的，所以应该效率也不低，不过暂时只写了个没复用版本，时间复杂度 n^2，实际运行效率就比冒泡快一点 TAT，等着我以后来优化，目标优化到 nlogn 的时间复杂度。
